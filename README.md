# books-memorization.github.io

This is the code repository for the companion website to the paper
[*Extracting memorized pieces of (copyrighted) books from open-weight language models*](https://arxiv.org/abs/2505.12546)
(Cooper et al., COLM 2026). The site is live at
[books-memorization.github.io](https://books-memorization.github.io).

The site is static; `python3 build.py` regenerates it from the sources in `build-src/` and the
figures in `assets/`.
